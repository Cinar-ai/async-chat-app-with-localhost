# ASYNC - Real-time Chat App 

A minimalist, fast, and modern real-time messaging platform built with Node.js and Socket.io. Zero noise, just pure conversation.

##  Features
- **Real-time Communication:** Instant message delivery using Socket.io.
- **Modern UI:** Sleek, dark-themed, and responsive design.
- **Lightweight Database:** No complex setup required; uses a local JSON-based file system.
- **Guest Access:** Jump into conversations instantly without creating an account.

##  Installation & Setup
